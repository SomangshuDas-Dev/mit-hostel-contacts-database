# M.I.T Hostel Contacts Database

This repository contains the website files used for the **M.I.T Hostel
Contacts Database**.\
The project provides a simple web interface for accessing and
interacting with hostel contact information.

The repository is **public only for the purpose of GitHub Pages
hosting**.\
All files are maintained primarily for **personal use and internal
reference**.

------------------------------------------------------------------------

## Overview

The repository stores the static web resources required for the hostel
contacts system.\

Typical contents of this repository include:

-   HTML files for the user interface
-   CSS files for styling
-   JavaScript files for client-side functionality
-   Supporting assets required for the web interface

This repository does **not contain the actual database**.\
It only contains the **web pages and scripts used to access or interact
with the database**.
